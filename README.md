## O que é Big Data?

**Big Data** refere-se a conjuntos de dados tão grandes, complexos ou em crescimento tão rápido que ferramentas tradicionais de processamento e armazenamento se tornam insuficientes. O termo é comumente descrito pelos **5 Vs**: **Volume** (grande quantidade de dados), **Variedade** (dados estruturados, semiestruturados e não estruturados), **Velocidade** (geração e processamento em tempo real ou próximo disso), **Veracidade** (confiabilidade e qualidade dos dados) e **Valor** (extração de insights e resultados úteis). Big Data envolve tecnologias e técnicas para armazenar, processar e analisar esses dados em escala, como ecossistemas distribuídos (Hadoop, Spark), bancos NoSQL e linguagens como Python com bibliotecas de análise e processamento.

## Tópicos abordados

1. **Princípios de Big Data** — fundamentos, conceitos e ecossistema de Big Data.
2. **Hadoop e armazenamento de dados** — arquitetura Hadoop, HDFS e modelos de armazenamento distribuído.
3. **Spark com Python (PySpark)** — processamento distribuído com Apache Spark usando PySpark.
4. **Análise de dados em Python com Pandas** — manipulação e análise de dados com a biblioteca Pandas.
5. **Big Data Analytics** — atividade prática supervisionada (APS) de análise em Big Data.
